Publicering

Bortsett från studenter.miuns SFTP-server, har jag tidigare bara provat Render och Netlify. Därför valde jag att prova Github Pages för denna laboration. Github Pages kan hosta statiska webbplatser och har stöd för HTML, CSS och JavaScript. Direkt i repots inställningar kan man välja branch och mapp som ska publiceras. Därefter får man en länk till den publicerade webbplatsen.

Så länge webbplatsen publiceras publikt är Github Pages kostnadsfritt. Dock finns Github Enterprise som ger möjlighet att begränsa åtkomsten och göra den privat. För det finns en 30 dagars provperiod, därefter kostar det 21 USD /månaden (ca 200 sek) per användare.

För mig personligen är det hittills den lättast använda webhosten. Det gick snabbt och enkelt att publicera, utan större problem. Något som bidrog till det att publiceringen görs direkt i repots inställningar och inte kräver mycket konfigurering. Dock har den endast stöd för statiska sidor och hosten syns i länken. Därför skulle jag nog bara använda den för utbildningen och privata projekt.
