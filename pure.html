Pure

https://pure-css.github.io/start/

Pure är ett kostnadsfritt CSS-ramverk som erbjuder repsonsiva moduler. Modulerna är små i storleken och inkluderas genom att länka till stylesheeten i head-elementet. Pure CSS kan även inkluderas genom npm.

Ramverket är lättanvänt med tydlig dokumentation. Det är litet i storleken och man kan snabbt skapa en responsiv layout. Men genom att det är litet i storleken finns inte mycket möjlighet att anpassa stylingen, såsom att ändra färger, formulär eller tabeller. Det finns inte heller något inbyggd modul för att skapa en responsiv meny. Det innebär att man behöver skriva en del css själv och i slutändan har inte mycket tid sparats på styling. Eftersom det finns möjlighet att hämta moduler individuellt kan det dock vara användbart om man vill använda Pure CSS för exempelvis endast knappar eller formulär.
