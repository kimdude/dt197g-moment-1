Bootstrap

https://getbootstrap.com/docs/5.3/getting-started/introduction/

Bootstrap är ett kostnadsfritt ramverk för frontend-utveckling. Ramverket erbjuder responsiva komponenter för exempelvis knappar, tables, forms, navigation och mer. Den erbjuder även JavaScript plugins. Allt kan inkluderas antingen genom att länka i head-elementet eller npm.

Bootstrap är väldigt lättanvänt och går snabbt att komma igång med. Det sparar mycket tid att slippa skapa mediaqueries. Den har många komponenter varav majoriteten går att anpassa någorlunda utan egen CSS. Man kan skapa en sida utan att egentligen behöva skriva någon egen css-kod, om man inte vill.

Eftersom det finns en handfull alternativ för styling direkt med Bootstrap kan det dock synas att webbplatsen är skapad med ramverket. Det kan också skapa svårigheter om man vill skriva över Bootstraps styling. Dessutom är komponenterna stora. Behöver man endast CSS för exempelvis knappar finns ingen möjlighet att lägga till endast den komponenten. I så fall måste man använda SASS.
